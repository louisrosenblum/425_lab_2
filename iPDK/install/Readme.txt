HOW TO INSTALL AND USE PDK

Copy PDK folder to your PC

1. Add the content of .bashrc.user file to your .bashrc.user file in home directory
2. Copy cdesigner.tcl file to your home directory
3. Copy star-rcxt2010 file to your home directory
4. Copy lib.defs file in directory where you want to run Custom Designer
